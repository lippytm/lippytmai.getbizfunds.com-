# MyClaw Prompt Pack - Intake and Routing

## Purpose

Use these prompts to define how leads should be categorized, routed, and followed up inside MyClaw.

## Prompt 1 - Lead Categories

Create a practical lead categorization system for `lippytmai.getbizfunds.com` using these top-level categories:
- funding help
- AI setup
- automation services
- strategy session
- affiliate / ecosystem interest
- general inquiry

For each category, provide:
- intent description
- urgency level
- ideal follow-up type
- suggested next action

## Prompt 2 - Intake Form Design

Design a short, high-conversion intake form for the website that captures enough information to route the lead without overwhelming the visitor.

Include:
- essential fields
- optional fields
- routing logic notes
- trust-building form copy

## Prompt 3 - Workflow Routing Logic

Create a routing map for website leads.

For each lead category, define:
- where the lead goes
- what follow-up happens first
- what questions should be asked next
- whether the lead is sales-ready, nurture-ready, or needs manual review

## Prompt 4 - Follow-Up Sequences

Create 3 short follow-up sequence outlines for:
- funding inquiries
- automation requests
- strategy session requests

Keep the flow simple, clear, and action-oriented.
