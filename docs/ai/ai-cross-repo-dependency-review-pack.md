# AI Cross-Repo Dependency Review Pack

## Purpose
Use this pack to help AI systems review dependencies between repos, detect weak links, and improve continuity across the wider ecosystem.

## Review Questions
- Which repos depend on this repo most?
- Which dependencies are visible enough?
- Which dependencies are hidden or weakly documented?
- Which handoffs are likely to break or confuse?
- Which repo relationships should be clarified first?

## Suggested Prompt Ideas
- ai-cross-repo-dependency-review-v1
- ai-hidden-dependency-review-v1
- ai-repo-relationship-clarity-review-v1
- ai-dependency-risk-review-v1
- ai-dependency-continuity-review-v1

## Best Practices
- improve visibility of dependencies before adding more complexity
- use dependency reviews to strengthen handoffs and reuse
- surface both strengths and weak spots
- connect dependency clarity to system resilience and customer/client continuity
