# Role-Based Autonomous Prompt Pack — Knowledge

## Purpose
Use these prompts to help AI systems improve standards clarity, canonical ownership, prompt reuse, and learning visibility.

## Prompts

### 1. ai-knowledge-canonical-review-v1
Review this knowledge repository and return:
1. strongest canonical assets
2. duplicated guidance risks
3. assets that should be the single source of truth
4. best next cleanup action

### 2. ai-knowledge-discoverability-review-v1
Review this repo's prompts, templates, and guides and return:
1. easiest assets to find
2. hidden high-value assets
3. best index or surface improvement
4. best next action for reuse visibility

### 3. ai-knowledge-learning-review-v1
Review this repo's learning and onboarding assets and return:
1. strongest reusable learning materials
2. missing learning support by role
3. outdated learning materials
4. best next improvement for practical learning value

## Best Practices
- keep knowledge prompts focused on reuse and clarity
- strengthen canonical ownership
- improve discoverability before adding more assets
