# Role-Based Autonomous Prompt Pack — Front Door

## Purpose
Use these prompts to help AI systems improve public routing, CRM entry quality, asset visibility, and cross-repo handoffs.

## Prompts

### 1. ai-frontdoor-routing-review-v1
Review this front-door repository and return:
1. strongest public entry paths
2. weakest routing paths
3. where users may get stuck
4. best next routing improvement
5. best related repo to surface more clearly

### 2. ai-frontdoor-crm-entry-review-v1
Review how this repo captures and routes relationship or CRM entry information and return:
1. missing intake fields or prompts
2. weak next-action handling
3. weak handoff context
4. best CRM improvement with highest leverage

### 3. ai-frontdoor-asset-visibility-review-v1
Review this repo's reusable prompts, templates, and standards assets and return:
1. most reusable assets
2. least visible assets
3. best asset to expose more clearly
4. best cross-repo reference to add

## Best Practices
- keep front-door prompts public-value focused
- improve clarity before adding complexity
- preserve relationship context for handoffs
