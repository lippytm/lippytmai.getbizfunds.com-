# Role-Based Autonomous Prompt Pack — Product

## Purpose
Use these prompts to help AI systems improve product clarity, delivery assets, implementation readiness, and reusable outputs.

## Prompts

### 1. ai-product-readiness-review-v1
Review this product repository and return:
1. scope clarity
2. strongest delivery assets
3. biggest implementation gap
4. best next product improvement

### 2. ai-product-asset-reuse-review-v1
Review this repo's product assets and return:
1. most reusable implementation assets
2. assets that should be surfaced more clearly
3. assets that are too local or too vague
4. best asset to standardize next

### 3. ai-product-handoff-review-v1
Review how this product repo connects with front-door and workflow repos and return:
1. handoff strengths
2. handoff weaknesses
3. missing context or references
4. best next improvement for product continuity

## Best Practices
- keep product prompts delivery-focused
- improve scope clarity before expanding assets
- expose reusable implementation assets clearly
