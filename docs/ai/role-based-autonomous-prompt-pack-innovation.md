# Role-Based Autonomous Prompt Pack — Innovation

## Purpose
Use these prompts to help AI systems improve experiment clarity, lesson capture, promotion decisions, and innovation visibility.

## Prompts

### 1. ai-innovation-lesson-capture-v1
Review this innovation repository and return:
1. strongest lessons learned
2. experiments with unclear outcomes
3. best lesson to promote into knowledge repos
4. best next cleanup action

### 2. ai-innovation-promotion-review-v1
Review this innovation repo and return:
1. experiments ready for promotion
2. experiments that should stay local
3. experiments that should retire
4. best destination repo role for promoted ideas

### 3. ai-innovation-visibility-review-v1
Review how this innovation repo surfaces maturity and next actions and return:
1. maturity labeling gaps
2. visibility gaps
3. strongest reusable experimental assets
4. best next improvement for scanability

## Best Practices
- capture lessons, not only ideas
- promote useful concepts before they become stranded
- keep maturity and next action visible
