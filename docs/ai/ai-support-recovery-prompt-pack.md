# AI Support Recovery Prompt Pack

## Purpose
Use these prompts to help AI systems improve support recovery, complaint handling, and trust restoration.

## Prompts

### 1. ai-support-issue-summary-v1
Review this support case and return:
1. issue summary
2. urgency
3. what has already been tried
4. strongest next action
5. whether human escalation is needed

### 2. ai-support-recovery-plan-v1
Review this unresolved or negative support case and return:
1. likely source of friction
2. best corrective action
3. best owner for the next step
4. best wording for a clear recovery update
5. how success should be checked later

### 3. ai-support-trust-risk-review-v1
Review this support history and return:
1. trust risk signals
2. repeated confusion patterns
3. unresolved themes
4. best next action to reduce trust damage

## Best Practices
- summarize clearly before escalating
- use AI to shorten recovery time
- focus on clarity and corrective action, not generic apology language
