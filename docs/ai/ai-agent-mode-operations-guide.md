# AI Agent Mode Operations Guide

## Purpose

This guide defines how AI agents should operate across the wider ecosystem when reviewing, upgrading, documenting, routing, learning, and improving repositories, CRM layers, and reusable assets.

## Core Principle

AI agents should operate as structured helpers that improve:
- clarity
- reuse
- continuity
- documentation quality
- customer and client experience
- standards alignment
- system health

They should not create unnecessary complexity, duplicate ownership silently, or produce vague upgrade noise.

## Agent Operating Pattern

### Step 1 — Inspect
Review repo role, purpose, files, status, roadmap, assets, prompts, CRM signals, or related docs.

### Step 2 — Diagnose
Identify gaps, drift, risks, weak handoffs, weak lifecycle continuity, duplicated assets, or weak visibility.

### Step 3 — Recommend
Produce the smallest high-leverage next actions with clear ownership and expected benefit.

### Step 4 — Document
Summarize findings in structured formats that humans and other AI systems can reuse.

### Step 5 — Promote
Move durable lessons, assets, or prompts toward stronger canonical homes when appropriate.

## Agent Types

### Repo Review Agent
Improves repo surfaces, standards coverage, and interoperability.

### CRM Review Agent
Improves lifecycle continuity, next-best actions, customer engagement quality, and satisfaction visibility.

### Prompt Review Agent
Improves prompt quality, naming, discoverability, and reuse.

### Asset Governance Agent
Improves asset visibility, canonical ownership, and cross-repo reuse.

### Fleet Visibility Agent
Improves rollups, status scanning, priority comparison, and governance review.

### Innovation Promotion Agent
Improves experiment capture, lesson reuse, and promotion into stronger long-term repos.

## Best Practices
- keep outputs structured
- prefer next actions over generic observations
- preserve canonical ownership visibility
- improve scanability for humans and AI
- promote durable lessons into stronger long-term repos
- use AI to reduce drift, duplication, and hidden confusion
