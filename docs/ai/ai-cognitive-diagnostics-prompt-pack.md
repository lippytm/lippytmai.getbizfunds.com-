# AI Cognitive Diagnostics Prompt Pack

## Purpose
Use these prompts to help AI systems diagnose clarity gaps, weak reasoning surfaces, weak summaries, weak next actions, weak documentation, and weak handoffs across the ecosystem.

## Prompts

### 1. ai-cognitive-clarity-review-v1
Review this repo, document set, CRM surface, or workflow and return:
1. what is clearest
2. what is hardest to understand
3. what is missing for confident continuation
4. what summary should exist but does not
5. best next action to improve clarity

### 2. ai-documentation-transparency-review-v1
Review the visible docs and return:
1. strongest transparency surface
2. weakest transparency surface
3. hidden ownership gaps
4. missing scanability elements
5. best next documentation improvement

### 3. ai-handoff-diagnostics-review-v1
Review the handoff from one layer or repo to another and return:
1. strongest handoff element
2. missing context
3. weak continuity signal
4. trust or confusion risk
5. best next handoff improvement

### 4. ai-next-action-diagnostics-v1
Review the visible next actions in this system and return:
1. best visible next action
2. missing next actions
3. vague next actions
4. highest-risk ambiguity
5. best next action rewrite

### 5. ai-cognitive-drift-review-v1
Review this system over time and return:
1. signs of documentation drift
2. signs of standards drift
3. signs of lifecycle drift
4. signs of ownership drift
5. most important correction first

## Best Practices
- diagnose clarity before adding more systems
- improve summaries and next actions before adding more prompts
- treat weak handoffs as cognitive risk, not just documentation risk
- keep outputs structured and comparable
