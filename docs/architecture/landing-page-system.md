# Landing Page System

## Goal

Create a flexible landing page structure that allows different offers, audiences, and campaigns to have their own focused conversion path.

## Why This Matters

A homepage should introduce the platform, but landing pages should do the deeper conversion work for specific visitor intents.

## Recommended Landing Page Types

### By Offer
- funding help
- AI setup
- automation services
- strategy sessions
- affiliate / ecosystem participation

### By Audience
- startup founders
- solo business owners
- service businesses
- established business operators
- AI-curious businesses

### By Traffic Source
- Facebook ads
- Instagram campaigns
- chatbot referrals
- partner referrals
- email campaigns

## Standard Landing Page Structure

1. Clear headline
2. Value-focused subheadline
3. One main CTA
4. Problem / solution section
5. Benefits
6. How it works
7. FAQ
8. Trust section
9. Final CTA

## Best Practices

- one main goal per page
- one primary CTA per page
- match the page message to the traffic source
- keep copy specific to the visitor intent
- avoid sending paid traffic to a generic homepage when a focused landing page would convert better

## Reusable Assets

Build reusable blocks for:
- headline formulas
- CTA formulas
- FAQ patterns
- testimonials later on
- offer summaries
- trust builders

## Future Expansion

This system can support:
- A/B testing
- persona-based funnels
- offer-specific campaigns
- affiliate traffic segmentation
- social ad campaign routing
