# Premium Path Review Rubric

This rubric helps review premium-path changes in `lippytmai.getbizfunds.com-`.

## Review Dimensions
- premium offer clarity
- strategy-to-premium transition quality
- routing quality
- assistant or swarm handoff quality
- activation readiness
- public trust impact

## Best Practices
- keep premium paths easy to understand
- review premium routing against real service fit
- protect high-value handoff paths from confusion
- treat premium conversion quality as both trust and revenue signal
