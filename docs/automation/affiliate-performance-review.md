# Affiliate Performance Review

This document explains how to review affiliate and partner-origin performance in `lippytmai.getbizfunds.com-`.

## Goal

Measure affiliate and partner paths by quality, activation, and repeat value instead of only raw lead volume.

## Review Signals
- partner-origin lead quality
- premium-path conversion quality
- checkout and activation success
- follow-up timing quality
- repeat engagement and expansion signals

## Best Practices
- review affiliate sources by fit, not only by traffic count
- protect premium and partner-origin routing quality
- improve weak partner paths before scaling volume
- use repeat value and activation quality as core partner metrics
