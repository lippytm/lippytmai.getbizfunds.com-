# Lead Lifecycle

This document defines a simple CRM-style lead lifecycle for `lippytmai.getbizfunds.com-`.

## Stages

1. new_lead
2. engaged_lead
3. qualified_lead
4. strategy_lead
5. premium_lead
6. active_service
7. nurture_hold
8. reengagement
9. archived

## Best Practices

- keep stages easy to audit
- keep stage names simple
- tie stage changes to events
- keep service activation tracked clearly
