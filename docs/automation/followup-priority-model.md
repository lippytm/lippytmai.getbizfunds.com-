# Followup Priority Model

This document defines a simple priority model for follow-up work in `lippytmai.getbizfunds.com-`.

## Priority Bands

- urgent
- high
- normal
- low

## Suggested Signals

- urgency selected in intake
- stage in lifecycle
- premium or strategy fit
- time since last touch
- service activation status

## Best Practices

- prioritize premium and urgent paths carefully
- do not let old leads block time-sensitive leads
- keep follow-up rules understandable
- use this model to guide assistant and swarm routing
