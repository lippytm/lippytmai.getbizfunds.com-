# ASSETS

## Purpose
Document the reusable experiment, prototype, and lesson assets this innovation repository owns or references.

## Asset Inventory
| Asset Name | Category | Format | Status | Source | Notes |
|---|---|---|---|---|---|
| prototype_prompt_set | prompts | md | draft | local | experimental prompt collection |
| lesson_summary_pack | guide | md | draft | local | captures useful learning from tests |
| concept_validation_notes | report | md | active | local | helps decide promotion or retirement |

## Reusable Assets
- prototype prompts
- lesson summaries
- validated concept notes
- experiment tracking patterns

## Imported Assets
- opportunity signals from other repos
- standards references
- practical implementation constraints

## Asset Rules
- keep experiments labeled by maturity
- expose useful lessons quickly
- move validated assets into stronger long-term repos
- avoid leaving prototype assets undocumented
